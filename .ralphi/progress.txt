# Zvec CLI Development Progress

## Codebase Patterns

- **CLI Structure**: Commands are in `src/commands/`, each exports `parse*Args`, `show*Help`, and the main function
- **Config Pattern**: Use `ZVEC_HOME` env var for testing (overrides `os.homedir()`)
- **Bun File API**: Use `Bun.file()` for existence checks, `Bun.write()` for writing
- **Testing**: Use `bun:test` with `describe/test/expect`, set `ZVEC_HOME` for isolated tests
- **Imports**: Biome auto-sorts imports; use `node:` prefix for Node builtins
- **Spawn**: Use `Bun.spawnSync()` for shell commands like `mkdir -p`
